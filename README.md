# Cloud Monitoring Dashboard

This is a beginner-friendly Python project with automated testing and CI/CD using GitHub Actions.

## Features
- Simple Python application
- Unit testing with pytest
- Git version control
- GitHub Actions workflow for continuous integration

## Tech Stack
- Python
- pytest
- Git
- GitHub Actions

## Project Structure
- `app.py` - main Python app
- `test_app.py` - test file
- `requirements.txt` - dependencies
- `.github/workflows/python-app.yml` - CI workflow

## How to Run
```bash
python app.py
